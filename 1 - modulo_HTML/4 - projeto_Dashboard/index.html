Módulos

HTML

HTML (HyperText Markup Language) é a linguagem de marcação responsável por estruturar o conteúdo de uma página web — títulos, parágrafos, links, imagens e formulários. Ele não define aparência nem comportamento, apenas organiza e dá significado ao conteúdo.

Como no exemplo abaixo:

                    
                        <article>
                        <h2>Título do Artigo</h2>
                        <p>Conteúdo do parágrafo.</p>
                        </article>

CSS

CSS (Cascading Style Sheets) é a linguagem responsável pela aparência visual da página — cores, espaçamento, layout e responsividade. Ele separa o "como se parece" do "o que é", permitindo estilizar o mesmo HTML de formas completamente diferentes.

Como no exemplo abaixo:

                    
                        article {
                            display: flex;
                            flex-direction: column;
                            gap: 12px;
                            padding: 16px;
                            }

JavaScript

JavaScript é a linguagem de programação que dá interatividade e lógica à página — manipula elementos, responde a cliques e eventos, e consome dados de APIs em tempo real. É o que transforma uma página estática em uma aplicação dinâmica.

Como no exemplo abaixo:

                    
                        document.querySelector("button").addEventListener("click", () => {
                            alert("Botão clicado!");
                        });
