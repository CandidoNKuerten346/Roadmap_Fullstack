# Roadmap_Fullstack
Olá, bem-vindo ao meu projeto de formação, rumo aos conhecimentos consolidados do Fullstack Júnior!

Este repositório documenta minha jornada de estudo e prática — do zero até a construção de um sistema fullstack completo — combinando teoria (freeCodeCamp), prática guiada (missões/roadmap) e projetos reais aplicados a cada etapa.

## Objetivo do Projeto
O objetivo do projeto Roadmap_Fullstack é aprimorar tanto meus conhecimentos técnicos quanto meu portfólio, em busca da formação como desenvolvedor Fullstack Júnior — com cada módulo gerando projetos reais, publicados e documentados.

### Fases do Projeto
O projeto consiste em 6 módulos com conteúdos específicos por segmento, organizados por ordem de dependência técnica:

1. Módulo - HTML
    Fundamentos estruturais da web: semântica, formulários, acessibilidade e boas práticas. Foco em aprendizado total sobre HTML antes de qualquer estilização ou interatividade.

2. Módulo - CSS
    Estilização e responsividade: box model, Flexbox, Grid, mobile-first. Todos os projetos do módulo de HTML são revisitados e estilizados nesta etapa.

3. Módulo - JavaScript
    Lógica de programação, manipulação de DOM, consumo de APIs e fundamentos de ES6+. Base para tudo que vem depois no front-end.

4. Módulo - React
    Componentização, hooks, gerenciamento de estado e roteamento. Construção de interfaces dinâmicas consumindo APIs externas.

5. Módulo - Backend
    Node.js, Express, banco de dados, autenticação e construção de APIs RESTful profissionais.

6. Módulo - Projeto Final
    Sistema fullstack completo (SaaS), integrando front-end e back-end, com deploy, containerização (Docker) e publicação em cloud.
